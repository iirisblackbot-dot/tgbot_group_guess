# 🎮 Игровой Telegram-бот

Многофункциональный Telegram-бот с 6 играми, написанный на Python.

## 🎯 Игры

| Игра | Описание |
|------|----------|
| ⚔️ Крестики-нолики | Игра против умного бота (минимакс алгоритм) |
| ✂️ Камень-ножницы-бумага | Классическая игра против бота со счётом |
| 🚢 Морской бой | Поле 7×7, умный бот, расстановка кораблей |
| 🎰 Казино (слоты) | Слот-машина с монетами, ставками и бонусами |
| ⛏️ Майнкрафт | Добыча ресурсов, крафт предметов, уровни |
| 🎨 Рисование (AI) | Генерация изображений через DALL-E 3 |

## 📦 Установка

### Требования
- Python 3.11+
- pip

### Установка зависимостей

```bash
pip install python-telegram-bot==20.7 openai pillow requests aiohttp
```

## ⚙️ Настройка

1. Создай бота через [@BotFather](https://t.me/BotFather) и получи токен
2. Открой файл `config.py` и вставь свой токен:

```python
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
```

3. Для работы генерации изображений (🎨 Рисование) нужен OpenAI API ключ:

```bash
export OPENAI_API_KEY="ваш_ключ_openai"
```

Или вставь в `config.py`:
```python
OPENAI_API_KEY = "ваш_ключ_openai"
```

И в `games/drawing.py` замени строку:
```python
api_key=os.environ.get("OPENAI_API_KEY", ""),
```
на:
```python
api_key="ваш_ключ_openai",
```

## 🚀 Запуск

```bash
# Обычный запуск
python3 bot.py

# Или через скрипт
./start.sh

# Запуск в фоне (Linux)
nohup python3 bot.py &
```
